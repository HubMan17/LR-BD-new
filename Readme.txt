---Установка---
1. Запустить файл Setup.exe
2. При установки Python выбрать пункт "Add Python X.X to PATH"
3. После завершения установки запустить файл Setup.exe
4. Успешно использовать программу :)

---Запуск программы---
1. Запустить файл main
2. Выбрать файл базы данных нужной структуры
3. Ввойти в профиль
   *Базовый профиль админа:
    login: admin
    password: 123
